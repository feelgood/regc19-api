# C19 Symptoms registration API

## Development
```
./start.sh
```

or

```
virtualenv venv
. venv/bin/activate
pip3 install -r requirements.txt
export FLASK_APP=regc19api
export FLASK_DEBUG=1
flask run -p 5000
```

## Usage

An OpenAPI specification will be published later

GET, POST, PUT and DELETE `/c19/registrations`
